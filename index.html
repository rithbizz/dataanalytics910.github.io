def describe(data, label="Dataset"):
    data = sorted(data)
    n = len(data)
    mean = statistics.mean(data)
    med = statistics.median(data)
    try:
        mode_val = statistics.mode(data)
    except:
        mode_val = "None (all unique)"

    variance = statistics.variance(data)

Summary: Descriptive Statistics

What youâ€™ve learned today:

Category Measures Best Used When
Central Tendency Mean, Median, Mode Symmetric / Skewed / Categorical
Dispersion Range, Variance, Std Dev, IQR Quick view / Precise / Outlier-robust

ğŸ¯ Key Takeaways

  • Mean = fair average, sensitive to outliers
  • Median = middle value, robust to outliers
  • Mode = most common, works for categories
  • Std Dev = typical distance from mean, same units as data
  • IQR = spread of middle 50%, unaffected by outliers
  • Always pair central tendency with dispersion â€” one without the other is incomplete!
  • When in doubt: check for outliers first, then choose your measures

Common Mistakes to Avoid

âŒ Reporting only the mean without any dispersion measure
   â†’ The mean of 75 means nothing without knowing the spread!

âŒ Using mean with heavily skewed or outlier-affected data
   â†’ Use median instead for income, house prices, wait times

âŒ Confusing variance and standard deviation units
   â†’ Variance is in squared units; std dev is in original units

âŒ Forgetting to sort before finding the median
   â†’ Median of unsorted data is meaningless!

âŒ Assuming mode always exists
   â†’ If all values are unique, there is no mode

âŒ Ignoring outliers without investigating them
   â†’ Outliers are often the most interesting and important data points!

Quick Reference Card

\[\bar{x} = \frac{\sum x_i}{n} \qquad \text{(Mean)}\]

\[\text{Median} = \text{middle value of sorted data}\]

\[s^2 = \frac{\sum(x_i - \bar{x})^2}{n-1} \qquad \text{(Variance)}\]

\[s = \sqrt{s^2} \qquad \text{(Standard Deviation)}\]

\[\text{IQR} = Q_3 - Q_1\]

\[\text{Outlier fences: } Q_1 - 1.5 \times \text{IQR} \quad \text{and} \quad Q_3 + 1.5 \times \text{IQR}\]
